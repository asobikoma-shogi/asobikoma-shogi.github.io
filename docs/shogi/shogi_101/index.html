日本將棋 101

大家好，我們是CrossingVoid與Quisette。 相信大家點開這份筆記一定是對將棋有興趣吧！ 將棋在台灣非常小眾，就算別人來問，也只會說：「喔喔，象棋喔」😵‍😵‍ 其實我們覺得台灣人裡有很多人喜歡日本文化，而將棋就是一個很好的認識深日本之途徑。 只是因為將棋資源取得不易，而絕大多數都是日文，導致很多朋友還沒學會最基本的規則就先放棄了。這樣是非常可惜的，還沒探索到真正好玩的地方，便打退堂鼓，則失去了一個可能伴你一生興趣的機會！ 有鑑於此，在此希望用我們的將棋學習歷程，讓想了解、學習將棋的朋友少走一些冤枉路！🚀

  1. 對局： 對局(たいきょく)，一場將棋遊戲，也可以當作動詞，進行對局↩
  2. 反則手：反則(はんそく)，違反規則的意思。所以反則手為下出違反規則棋步。,在正常對局中，出現反則手的一方判負並結束該對局。↩
  3. 戦法：戦法(せんぽう)，為進攻的陣形。在將棋中，有多達百種的戰法，每個戰法所攻擊的目標皆有所不同，初學者可以先選擇其中一種戰法學習↩
  4. 囲い：囲い(かこい)，翻成中文為圍玉，玉指玉將。因為在將棋中，玉將被捕獲就輸掉對局，所以一般情形會用其它棋駒把玉將圍住，保護玉將，稱為圍玉↩
